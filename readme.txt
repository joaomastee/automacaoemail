1.Faça o upload do arquivo PHP para seu site hospedado na Hostinger. Isso pode ser feito usando o gerenciador de arquivos no painel de controle da sua conta, ou usando um cliente FTP como o FileZilla.
2.Crie o banco de dados MySQL na sua conta Hostinger e importe a tabela de destinatários. Isso pode ser feito usando o painel de controle de banco de dados ou usando um gerenciador de banco de dados como o phpMyAdmin.
3.Edite as configurações de conexão com o banco de dados no arquivo PHP. Insira o nome do banco, usuário e senha, e o endereço do banco de dados.
4.Edite as configurações de e-mail. Insira o endereço de e-mail de onde os e-mails serão enviados, assunto, e corpo do e-mail.
5.Faça o upload do arquivo CSV com a lista de destinatários para o seu site.
6.Execute o script PHP no navegador. Certifique-se de que o script esteja sendo executado corretamente e verifique se os e-mails estão sendo enviados corretamente.
7.(Opcional) Configure o agendamento de tarefas para executar o script automaticamente. Isso pode ser feito usando o Cron Jobs no painel de controle da sua conta Hostinger.
